# Code of Conduct

## 我们的承诺

为了营造一个开放、包容的社区环境，我们承诺让每个参与者都能受到尊重，不受骚扰。

## 我们的标准

鼓励的行为：
- 使用友好、包容的语言
- 尊重不同观点和经验
- 建设性地接受批评
- 关注社区最佳利益

禁止的行为：
- 使用性别化或其他歧视性语言
- 人身攻击或政治性攻击
- 公开或私下骚扰
- 未经明确许可发布他人私人信息

## 我们的责任

项目维护者负责明确可接受行为的标准，并对不可接受行为采取适当、公平的纠正措施。

## 适用范围

本行为准则适用于所有项目空间和代表项目的公共场合。

## 执行

辱骂、骚扰或其它不可接受的行为可向项目维护者举报。所有投诉将被审查和调查。

## 归属

本行为准则改编自 [Contributor Covenant](https://www.contributor-covenant.org) 2.1 版。
